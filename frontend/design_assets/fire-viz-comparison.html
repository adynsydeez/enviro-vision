Project Pyro — Fire Visualization Options

Same 40×40 cellular automata simulation, four different rendering approaches. Running live.

Option 1 Recommended
HTML5 Canvas + Shadow Glow
Custom Leaflet layer. Per-cell glow via ctx.shadowBlur. Fire colour shifts red→yellow based on burn age.
Full glow control ~2ms render Scales to 500×500 ~30 lines boilerplate
Option 2
DOM / SVG Rectangle Grid
One <div> per cell. Easy click handlers but flat visual, no blur effects, poor update performance.
Native click events No glow / blur Jank at 100×100 Flat appearance
Option 3
Canvas Multi-layer Particles
Simulates the visual ceiling of deck.gl/WebGL — radial gradients, floating sparks, layered blending. More code to maintain.
Highest visual quality Sparks + layered glow ~8ms render Complex to maintain
Option 4
Offscreen Canvas → Image Overlay
Renders to an offscreen canvas, converts to dataURL, sets as <img src>. Simplest Leaflet integration, but conversion overhead and visible update lag.
Trivial Leaflet setup toDataURL() ~12ms Visible update flash GC pressure
